# 海外文物知识服务子系统 - Web端

## 项目简介

本项目是海外文物知识服务子系统的Web前端部分，基于知识图谱构建子系统获取的数据，为用户提供海外文物的浏览、查询、可视化与个性化服务。界面设计参考克利夫兰艺术博物馆、大英博物馆等国际一流博物馆网站。

## 技术栈

| 技术 | 说明 |
|------|------|
| 前端框架 | Vue 3 |
| 构建工具 | Vite |
| UI组件库 | Element Plus |
| 可视化库 | ECharts / AntV G6 |
| HTTP客户端 | Axios |
| 状态管理 | Pinia |
| 路由 | Vue Router |

## 功能模块

### 1. 数据浏览

- **列表与卡片视图切换**：支持在列表视图与图片卡片视图之间切换，满足不同用户浏览习惯
- **多维筛选与排序**：支持按文物类型、年代、所属博物馆、材质等维度进行组合筛选；支持按名称、年代、更新时间等字段排序
- **文物详情页**：完整展示文物的文本描述、高清图片、三元组属性等信息；图片支持缩放、全屏查看；显示文物在知识图谱中的关联实体
- **相关文物推荐**：基于相似主题、同一作者、同一朝代、图像内容相似度等规则推荐相关文物
- **文物对比功能**：支持用户选择2-3件文物进行横向属性对比，直观展示差异

### 2. 数据查询

- **简单查询**：支持按文物名称、博物馆名称、年代等关键字进行全文检索，结果页支持分页与二次筛选
- **高级查询**：支持对文物类型、年代范围、材质、收藏地等多个字段进行组合限定查询
- **知识图谱查询**：支持用户以自然语言或结构化方式查询实体间的关系，如“查询某朝代的所有瓷器”或“查询某位艺术家的全部作品”
- **查询结果导出**：支持将查询结果导出为CSV或JSON格式

### 3. 数据可视化

- **知识图谱关系图**：采用力导向图展示文物实体及其关联关系，支持节点点击展开、悬停显示属性、缩放与拖拽交互
- **文物时间轴**：以时间轴形式展示各历史时期的文物分布，支持按朝代或自定义时间段筛选，点击时间节点可跳转至对应文物列表
- **文物地理分布图**：以地图形式展示文物现藏博物馆的地理位置，支持点击地图标注查看该博物馆的藏品信息
- **统计分析看板**：以图表形式展示文物数量、类型占比、朝代分布、博物馆藏量等统计数据

### 4. 用户个人信息管理

- **基础功能**：用户注册与登录，支持设置用户名、密码、头像等个人信息
- **浏览记录**（选做）：自动记录用户最近浏览的文物，支持一键清除
- **收藏功能**（选做）：支持用户收藏感兴趣的文物，收藏夹支持分组管理
- **评论与互动**（选做）：支持用户对文物发表评论，支持点赞与回复
- **个性化推荐**（选做）：根据用户浏览与收藏记录，在首页推送可能感兴趣的文物
